## What is EssentialUtils?

EssentialUtils is a **Folia-optimized survival utilities plugin** for Paper/Folia 1.21.8+. It provides quality-of-life features that enhance the survival experience without feeling like cheats.

## Features at a Glance

| Feature | Description | Activation |
|---------|-------------|------------|
| **Tree Feller** | Chop entire trees with one swing | Crouch + break log |
| **Vein Miner** | Mine connected ore veins | Break any ore |
| **Auto Farm** | Harvest and replant crops in an area | Break mature crop |
| **Chunk Loader** | Keep farm chunks loaded offline | Crouch + break crop |
| **Tab Menu** | Animated player list with stats | Always active |

## Requirements

- **Paper 1.21.8+** or **Folia**
- **Java 21** (server must run on Java 21)

## Quick Start

1. Download the latest release from [GitHub Releases](https://github.com/cryptofyre/EssentialUtils/releases)
2. Place the JAR in your `plugins/` folder
3. Start your server
4. Edit `plugins/EssentialUtils/config.yml` as needed
5. Run `/eutils reload` to apply changes
